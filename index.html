About

UW Exam Scheduler is a full-stack web application designed for University of Waterloo students to quickly search for their final exam details and sync them directly to their Google Calendar. Instead of manually searching through exam schedule PDFs and creating calendar events, students can find their exams in seconds and export them with one click.

The application scrapes official exam schedule data from the University of Waterloo website, processes it into a searchable format, and serves it through a clean REST API. The React frontend provides an intuitive search interface with real-time filtering as users type.

How It Works

Students sign in with their Google account, search for their course codes (e.g., "CS 246"), and view complete exam details including date, time, and location. With one click, they can create a Google Calendar event with all the exam information pre-filled, ensuring they never miss an important exam.

Note: This application is currently non-functional. The University of Waterloo updated their system to require authentication for accessing exam schedule data, which broke the public data scraping functionality.

Key Features

  • Instant Search: Real-time course code search with results appearing as you type
  • Google Calendar Integration: One-click export of exam details to your personal calendar
  • OAuth Authentication: Secure Google sign-in for calendar access
  • Automated Data Pipeline: Python scripts scrape and process official UW exam schedules

Technical Architecture

Frontend

  • React 18 with Vite for fast development and HMR
  • React Router for client-side navigation
  • Google API client (gapi) for Calendar integration
  • Responsive CSS with custom component styling

Backend

  • Express.js server with RESTful API endpoints
  • CORS-enabled for frontend communication
  • JSON-based exam data storage

Data Pipeline

  • Python scripts using Pandas for HTML table parsing
  • Automated scraping from official UW exam schedule pages
  • Data transformation and unique ID generation

Development Highlights

  • Built complete OAuth flow for secure Google Calendar access
  • Implemented fuzzy search with debouncing for responsive user experience
  • Created reusable modal components for exam details and confirmations
  • Designed alert system for user feedback on calendar sync operations
  • Developed Python data pipeline to parse and transform exam schedule HTML tables
